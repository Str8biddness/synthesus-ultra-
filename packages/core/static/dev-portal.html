Platform Overview

What is SYNTHESUS?

SYNTHESUS is a production-grade AI character intelligence platform built by AIVM. It enables developers to create, deploy, and manage synthetic characters that can engage in natural conversations, maintain persistent memory, and exhibit personality-driven behaviors.

Quick Start

  1. Generate an API key from the API Keys tab
  2. Include your key in the X-API-Key header
  3. Query /api/v1/characters to list available characters
  4. Send messages via /api/v1/query endpoint

Authentication

API Key Authentication

All API requests require authentication via the X-API-Key header. API keys are tied to usage tiers with different rate limits and features.

curl -H "X-API-Key: syn_your_api_key_here" \\ https://api.synthesus.ai/api/v1/health

Tiers & Rate Limits

Tier Rate Limit Features
Standard 60 req/min Basic characters, single-turn queries
Premium 300 req/min Multi-turn sessions, telemetry access
Enterprise 1000+ req/min Custom characters, SLA, dedicated support

API Endpoints

GET /api/v1/health

System health status and metrics

{ "status": "healthy", "version": "2.0.0", "rag": { "vectors": 78432 }, "characters_loaded": 10, "world_architecture": { "enabled": true, "active": true } }
GET /api/v1/characters

List all available characters

curl -H "X-API-Key: your_key" \\ https://api.synthesus.ai/api/v1/characters
POST /api/v1/query

Send a message to a character

curl -X POST \\ -H "X-API-Key: your_key" \\ -H "Content-Type: application/json" \\ -d '{ "query": "Tell me about quantum computing", "character": "synth", "mode": "auto", "session_id": "uuid-here" }' \\ https://api.synthesus.ai/api/v1/query
POST /api/v1/feedback

Submit feedback for response quality

{ "session_id": "uuid", "query": "original question", "response": "character response", "source": "cognitive", "rating": 5 }
